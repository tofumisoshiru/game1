小学生のためのプログラミング教材の説明

#目的
小学生がプログラミングを学ぶ第一歩として
とりあえずプログラムに触れてみるためのもの

#狙い
・ルールに従って何か書いたらルール通り動く
　というのを体感してもらう。
・直感的に楽しく体感してもらうために簡単な
　シューティングゲームを作った。そこに機能を
　追加していくという流れでプログラミングを
　体験してもらう。
・ビジュアルプログラミングではなく高級言語
　を扱うことにこだわっている。ビジュアル
　プログラミングと高級言語の橋となるような
　教材を目指している。
・ターミナルや長いコードを見せてしまうと
　プログラミングが難しいという印象を与えて
　しまうため、Web上だけで操作できるように
　した。これは、元のコードを書き換えられて
　システムが壊れてしまうのを防ぐという目的
　も兼ねている。
・HTMLとJavaScriptとCSSのみで構成されている。
　特殊な環境構築をしなくても、URLを踏むだけで
　使えるという手軽さを求めた。

#講座の流れ
0. 最初は普通に遊んでもらう。何か変更したいことや
　追加したいものなど改善案を挙げてもらう。
　ゲームをよりよくするために機能を追加していく。

1. まずは、文字列表示(printf)から。的をすべて壊した
　ときに何も起きないと面白くないから、的をすべて
　壊したときに自分で書いた文字列が表示されるようにする。
　再読み込みボタンでWebページをリセットしてから、
　printfのテキストボックスに何か好きな言葉を書いてもらう。
　例として提示するのは"GameClear"等
　書いたら実行ボタンを押して、的をすべて壊してみる。
　自分が書いた文字列が表示される。
　文字列表示の簡単な説明をする。

※今後は、再読み込みボタンをを押すたびに、前に書いた
　ものは書き直すようにする。再読み込みボタンを押すと
　完全にリセットされてしまうため、書き直さないと反映
　されなくなってしまう。
　オートコンプリートはONにしてあるので、一度書いて
　実行したものはテキストボックスにカーソルを合わせると
　簡単に再入力できるようになっている。
　オートコンプリートの履歴を消す方法
　URL:https://ageo-soft.info/knowhow/3527/

2. 数値関連の扱い。スコアが-1されていくのは変なので
　変更を加える。
　再読み込みボタンでWebページをリセットしてから、
　数値計算のところのテキストボックスに好きな数字を
　入れてもらう。
　例として提示するのは"100"等
　書いたら実行ボタンを押して、的をすべて壊してみる。
　スコアに自分の書いた数字が足されていくのがわかる。
　数値計算の簡単な説明をする。

3. 条件文(if文)の使い方。このままでは味気ないので
　的をすべて壊したときにエフェクトが表示される
　ようにする。
　再読み込みボタンでWebページをリセットしてから、
　if文のテキストボックスに0を書いてもらう。
　これは、条件として的が0個になったら動作するという
　説明を入れる。
　書いたら実行ボタンを押して、的をすべて壊してみる。
　花火が1つ上がる。
　if文の簡単な説明をする。

4. ループ文(for文)の使い方。花火が1つしか上がらないと
　華やかさが足りないので、花火の数を増やす。
　再読み込みボタンでWebページをリセットしてから、
　for文のテキストボックスに好きな数を書いてもらう。
　例として提示するのは"30"等
　これは、書いた数字の回数だけ繰り返し動作するという
　説明を入れる。
　書いたら実行ボタンを押して、的をすべて壊してみる。
　花火が書いた数字の回数分上がる。
　for文の簡単な説明をする。

以上のように、ゲームに機能を追加していくという流れで
プログラミングに触れてもらう。
